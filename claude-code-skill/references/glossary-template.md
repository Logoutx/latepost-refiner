# 统一校对表 模板

通读完全部转录后，照此填一张表，写到 `<输出目录>/校对表.md`（与成稿一起存档），供所有精校步骤共享，保证多份文件改得一致。这张表是“交叉验证 + 按主题核实”的产物。

## 人名（各种写法 → 统一为 + 身份/依据）
> 规则：能在转录内部互证或经公开资料核实的，写出“统一名”并标依据；拿不准的保留`（音）`或`（音，存疑）`，不臆造。

## 需特别修正 / 标注的转写错误
- 同音字错（例：真诚→臻臣、华海路→淮海路）：
- 英文听写错（例：CSE→3CE、亿特利→莹特丽）：
- 夹在文中的时间戳（如 `（09:02）`）：一律删除。
- 夹杂的英文乱码（如 `（English，34:47）`）：能判断词义就替换为正确中文/英文，判断不了就顺掉，不留乱码。
- `（音）`标记：能确定就改正去标；不能确定就保留。
- 存疑且关乎事实的（如某人籍贯、某轮投资主体）：保留原文 + `（音，存疑）`，不臆改。

## 精校规范要点（拷贝给每个子代理）
1. 保持对话体，保留发言人标签。
2. 删口癖与重复，但不改语气风格与原意、不加观点。
3. 理顺可读性，保留有个性/有信息量的金句。
4. 按主题加 `##` 小标题，不篡改原意、不加原文没有的结论、**一律不编号**（一份 6–20 个）。源文件若已带记者/速记小标题，先问用户再决定保留或重做。
5. 严格按本表统一人名/品牌/术语；删时间戳与英文乱码。
6. 保留全部事实细节（数字/金额/时间/产品/工艺/渠道/观点）——精校不是摘要。
7. 规范发言人标注（采访方归记者、被访方旁白归“同事”、群访按角色）。
8. 抬头：`# 标题`（格式 `英文名（中文名）title`）+ 第二行斜体 `*采访者：…｜时间：…*`。
9. 中文排版：引号一律全角 “”（内层 ‘’），禁 ASCII 直引号与「」『』；数字用阿拉伯数字（十六→16、六七十 B→60-70B），口语小数目/成语例外；中文与英文/数字间加一个半角空格（GPT-4、16 个、80%、2021 年）。
10. 一次处理 ≤20 段、不在 topic 中间断开；长文件分多次接力写到结尾，大块写入、初次落笔写对、不回头逐字微改。
