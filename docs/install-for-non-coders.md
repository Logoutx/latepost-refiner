# 给非技术同事的安装指南（Mac）

这份指南假设你完全不写代码、也不熟悉终端。照着做就行；中途弹出的提示框下面都有解释，别慌。

## 第一步：先决定用哪个版本

| 版本 | 适合谁 | 要装什么 | 难度 |
|---|---|---|---|
| **双击 App（本地网页版）** | 想在自己电脑上跑、手里有一个 API key | 下载一个程序 + 装文档转换器 | 最简单 |
| **Claude Code / Codex** | 经常用、想最省钱（用订阅、不填 key） | 装 Claude Code 或 Codex + 装文档转换器 | 偏技术 |

> 拿不准就先用 **双击 App**：下载一个自带运行时的程序，双击打开、浏览器里填表就能用，不用装 Node、不碰终端。

## 第二步：装“文档转换器”（只有本地版才需要）

如果你的访谈稿是 Word（.docx）、PPT、Excel 或 PDF，工具得先把它们转成纯文字。装一次，以后都不用再管：

1. 在工具文件夹里找到 `install-converters.command` 这个文件。
2. **右键点它（或按住 Control 再点）→ “打开” → 再点一次 “打开”。** 第一次必须这样：macOS 默认会拦下没有签名的程序，直接双击会被挡住；右键打开就能放行。以后正常双击即可。
3. 会跳出一个黑色的终端窗口，自动装好，最后显示 “✅ All set”。大概几分钟，期间别关窗口。

它只装缺的东西，重复跑也不会出问题。装好的是 markitdown（负责 Word/PPT/Excel + 普通 PDF）。复杂 PDF（多栏排版、扫描件、财报那种）需要额外的 docling，窗口最后会告诉你怎么加。

> 这一步会自动下载它需要的运行环境（Python），你不用提前装任何东西，也不用懂它是什么。

## 常见卡点

- **弹窗说“打不开，因为来自身份不明的开发者”**：别点 “移到废纸篓”。改成 **右键 → 打开 → 打开**。从网上下载的程序第一次都这样，正常。
- **装完了，但运行时说 “command not found”**：把终端窗口关掉，重新开一个再试。新装的工具要开一个新窗口才认得。
- **装 Claude Code 的 skill 时找不到 `~/.claude` 文件夹**：它是隐藏文件夹。在 Finder 里按 `Shift + Command + .`（句号键），隐藏文件就显示出来了；再按一次又藏起来。
- **下载很慢或卡住不动**：基本是网络问题。换个网络，重新双击那个 `.command` 文件即可，已经装好的不会重复装。

## 还要准备的（光有转换器还不够）

转换器只负责读文件；真正干活要连 AI，所以你还需要其中一种：

- **Claude Code**：一个 **Claude 订阅**（Pro 或更高）。
- **Codex**：一个 **ChatGPT 订阅**。
- **双击 App（本地网页版）**：一个 **API key**——找把工具分享给你的人要，填在网页里就行（只存在你本机，不会外传）。

---

有任何一步卡住，把终端窗口里的文字整段截图发给分享工具给你的人，比自己猜更快。
